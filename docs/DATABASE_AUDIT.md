# CyLaw.org — Database Audit

**Дата аудита:** 7 февраля 2026
**Источник URL-ов:** Sidebar `/common/left.html` на главной странице cylaw.org

---

## Методология

Боковая панель сайта (блок "Βάσεις Δεδομένων") загружается через jQuery из файла
`/common/left.html`. Она содержит ссылки на **все базы данных** сайта.

Для каждой пропущенной базы мы:
1. Зашли на главный индекс
2. Нашли все годовые/томовые подстраницы
3. Подсчитали уникальные ссылки `open.pl?file=...` на каждой подстранице

## ТЕХНИЧЕСКИЕ ЗАМЕТКИ

### Sidebar загружается через JS
```javascript
function loadContent() {
    $("#left").load("/common/left.html");
}
```
URL: `https://www.cylaw.org/common/left.html`

### Особенности структуры индексов

| Суд | Паттерн year index | Отличия от стандарта |
|-----|-------------------|---------------------|
| `areiospagos` | `index_{year}.html` | Стандартный |
| `apofaseised` | `index_{cat}_{year}.html` | 5 подкатегорий (pol/poin/oik/enoik/erg) |
| `jsc` | `index_{year}.html` | Стандартный, английский язык |
| `rscc` | `index_{volume}.html` | Тома (1–5), не годы; английский язык |
| `administrativeCourtOfAppeal` | `index_{year}.html` | Стандартный |
| `juvenileCourt` | `index_{year}.html` | Стандартный |
